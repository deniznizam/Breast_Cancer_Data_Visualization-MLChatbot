# Breast_Cancer_Data_Visualization-MLChatbot
Oracle altyapısının yüksek güvenlik ve performansıyla yönetilen verileri, gözetimli ve gözetimsiz makine öğrenmesi modelleriyle analiz eder. Temel farkı; karmaşık sayısal sonuçları, ileri veri görselleştirme teknikleriyle karar vericiler için şeffaf, anlaşılır ve aksiyona dönüştürülebilir stratejik içgörülere dönüştürmesidir
